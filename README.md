# Regeneron Pharmaceuticals

Regeneron Pharmaceuticals is a leading biotechnology company that invents, develops, and commercializes life-transforming medicines for people with serious diseases in eye disease, cancer, inflammation, infectious disease, and rare diseases. The Regeneron Genetics Center (RGC) has sequenced over 3 million human exomes and open-sources bioinformatics tools through its GitHub organizations.

**Website:** [https://www.regeneron.com](https://www.regeneron.com)
**Regeneron Genetics Center:** [https://www.regeneron.com/science/genetics-center](https://www.regeneron.com/science/genetics-center)
**GitHub (RGC):** [https://github.com/rgcgithub](https://github.com/rgcgithub)
**GitHub (MPDS):** [https://github.com/regeneron-mpds](https://github.com/regeneron-mpds)

## APIs

No public commercial developer APIs were identified for Regeneron Pharmaceuticals. Regeneron contributes open-source bioinformatics software including regenie, CLAMMS, and TCRAI via its GitHub organizations.

## Open Source Tools

| Tool | Description | Repository |
|------|-------------|------------|
| regenie | Whole genome regression for large-scale GWAS | [github.com/rgcgithub/regenie](https://github.com/rgcgithub/regenie) |
| CLAMMS | CNV detection from whole-exome sequencing | [github.com/rgcgithub/CLAMMS](https://github.com/rgcgithub/CLAMMS) |
| TCRAI | T cell receptor analysis and antigen mapping | [github.com/regeneron-mpds/TCRAI](https://github.com/regeneron-mpds/TCRAI) |
| ProPerMab | Antibody electrostatic potential analysis | [github.com/regeneron-mpds/propermab](https://github.com/regeneron-mpds/propermab) |
